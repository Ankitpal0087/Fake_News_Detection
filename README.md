# Fake_News_Detection
# Fake News Detection Using Machine Learning

## 📌 Project Overview
Fake News Detection is a Machine Learning based project that identifies whether a given news article is real or fake. 
The model analyzes the text content of news and predicts the authenticity of the information.

## 🎯 Objective
- To detect fake news using Natural Language Processing (NLP).
- To classify news articles as Real or Fake.
- To understand the application of Machine Learning in misinformation detection.
