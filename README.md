# Fake News Detection

## Project Description
Fake News Detection is a Machine Learning based project that identifies whether a news article is real or fake. The main goal of this project is to develop a classification system that can help detect misleading or false information from news data.

## Objective
The objective of this project is to build and evaluate different Machine Learning models for classifying news articles into real and fake categories.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Machine Learning Algorithms Used

This project uses multiple machine learning algorithms to classify news articles as real or fake.

Algorithms:
- Logistic Regression
- Random Forest Classifier
- Decision Tree Classifier
- Gradient Boosting Classifier

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Model Training
5. Model Evaluation
6. Fake News Prediction

## Features
- Classifies news articles as Real or Fake
- Uses multiple Machine Learning algorithms
- Compares model performance
- Provides prediction results

## Conclusion
This project demonstrates the use of Machine Learning algorithms for detecting fake news and shows how classification models can be applied to solve real-world problems.
